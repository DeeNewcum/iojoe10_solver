This is a solver for iojoe's game ["10"](http://www.iojoe.com/10/) [[2]](http://www.youtube.com/watch?v=R04gRJHkeXk).

Current status: Many boards can be solved within a few seconds, however, some boards tame an extremely long time to solve.  Work is ongoing to speed things up.
